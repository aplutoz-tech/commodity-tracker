# 📊 全球大宗商品價格追蹤儀表板

即時追蹤全球主要能源與金屬大宗商品的價格走勢。

## 🌟 功能特色

- 💹 即時報價：顯示目前價格、日漲跌幅
- 📈 歷史走勢圖：可選 1週 / 1月 / 3月 / 6月 / 1年
- 🔄 自動更新：每 5 分鐘自動重新整理
- 📋 原始數據：可查看並下載 CSV

## 🛒 追蹤商品

### ⚡ 能源
- WTI 原油、布倫特原油、天然氣

### 🥇 金屬
- 黃金、白銀、銅、鋁、鉑金

## ☁️ 部署到 Streamlit Cloud

1. 前往 https://share.streamlit.io
2. 用 GitHub 帳號登入
3. 選擇此 repo → main → app.py
4. 點 Deploy！

## 技術說明

| 套件 | 用途 |
|------|------|
| streamlit | 網頁儀表板框架 |
| yfinance  | 免費 Yahoo Finance 資料 |
| pandas    | 資料處理 |
| plotly    | 互動式圖表 |

> 資料來源為 Yahoo Finance 期貨合約，僅供參考，不構成投資建議。
